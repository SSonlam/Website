Data Provenance Tools

How is provenance supported? What methods/techniques/software tools (if any) do they use?

Git

  • Designed for code and software engineering
  • Promotes software reuse
  • Allows teams to work on the same code file, saving any changes made to the code

PRiME

  • Software engineering technique for adapting application designs to enable them to interact with a provenance middleware layer
D-PROV
  • A model in which provenance traces scientific workflow are captured and stored along with the data products
